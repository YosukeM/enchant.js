Class enchant.gl.Sprite3D

  • Defined in: gl.enchant.js
  • Extends enchant.EventTarget [/lang], enchant.EventTarget [/lang] [lang:en] Class with Sprite3D display function.

    Cube loaded by default.

    By adding {@link enchant.gl.Scene3D} instance, you can display atop an image. By changing {@link enchant.gl.Sprite3D#vertices}, {@link enchant.gl.Sprite3D#indices}, {@link enchant.gl.Sprite3D#normals} and others, you can freely draw Sprite3D, as well as pasting texture and more.

    In addition, it is also possible to add Sprite3D as a child, and all child classes will be drawn with coordinates based on their parents.

Class Detail

enchant.gl.Sprite3D()
[lang:ja] Sprite3D表示機能を持ったクラス.

デフォルトでは立方体がロードされる.

enchant.gl.Scene3Dのインスタンスに追加することで, 画面上に表示することができる. enchant.gl.Sprite3D#vertices, enchant.gl.Sprite3D#indices, enchant.gl.Sprite3D#normalsなどを変更することで, 任意のSprite3Dを描画することもでき, テクスチャなども貼付けることができる.

また, Sprite3Dを子として追加することも可能で, 子は全て親を基準とした座標系で描画される.

  //シーンの初期化
  var scene = new Scene3D();
  //Sprite3Dの初期化
  var sprite = new Sprite3D();
  //Sprite3Dをシーンに追加
  scene.addChild(sprite);
  //Scene initialization
  var scene = new Scene3D();
  //Sprite3D initialization
  var sprite = new Sprite3D();
  //Add Sprite3D to scene
  scene.addChild(sprite);
